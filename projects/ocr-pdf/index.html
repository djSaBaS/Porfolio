OCR para PDF

Repositorio

Qué problema resuelve

Permite recuperar texto en documentos PDF escaneados para editar, buscar y reutilizar información sin copiar manualmente.

Qué conseguí / impacto

Reducción de tiempos en tareas administrativas y menor riesgo de errores humanos.

Pipeline OCR reutilizable con manejo de incidencias y salida preparada para otros procesos.

Cómo funciona

Procesa el PDF, aplica OCR y devuelve texto estructurado para uso directo o integración.
